# 🚗 AutoMate Admin Panel

Professional admin panel for the AutoMate car service app — pure HTML/CSS/JavaScript, no build tools needed!

---

## 🚀 How to Run

### Option 1: Open Directly (Easiest)
1. Just **double-click `index.html`** to open in your browser
2. That's it! No installation needed

### Option 2: VS Code Live Server (Recommended)
1. Open the folder in **VS Code**
2. Install **"Live Server"** extension (by Ritwick Dey)
3. Right-click `index.html` → **"Open with Live Server"**
4. Browser opens automatically with live reload

## 📋 Features

✅ **Dashboard** — Stats cards, revenue chart, recent bookings
✅ **Bookings** — View all, filter by status, confirm/cancel/complete
✅ **Users** — All registered users, Pro/Free filter, search
✅ **Services** — Edit service prices, descriptions, status
✅ **Pro Members** — Manage Pro subscriptions, see savings
✅ **Vehicles** — All registered vehicles (cars & bikes)
✅ **Notifications** — Send push notifications, quick templates, history
✅ **Reports** — Revenue analytics, service performance, CSV export
